# NC.MicroService

This project is for learning microservices with .NET Core.

Based on 
- .NET Core 3.1 
- EFCore + MySql
- Consul：服务注册发现、配置中心
- Polly：熔断、限流
- Ocelot：网关
- IdentityServer4：认证授权
- ServiceComb Saga：分布式事务 *// TODO，事务出现异常时，补偿操作不能正常执行...*
- CAP：事件总线ESB
- SkyWalking + SkyAPM-dotnet：链路监控
- Serilog + Elasticsearch + Kibana：日志中心

***Keep Moving !!!***
